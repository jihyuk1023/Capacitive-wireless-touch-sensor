0. 준비하기
(1) 터치센서와 중간망 장치의 전원을 켭니다.

    *주의사항 : 터치센서는 전원을 키는 순간부터 약 3초 가량 주변의 전위를 인식하여 기본값으로 설정합니다.
    따라서 터치센서의 전원을 켤 때부터 3초까지는 터치센서의 철판을 만지지 않도록 합니다.
    터치센서를 켜는 순간에도 철판이 손이 닿아있지 않도록 주의해야 합니다. 기본값이 오염될 수 있기 때문입니다. 
(2) <data> 폴더에 있는 <나눔스퀘어라운드 ExtraBold> 폰트를 더블클릭하여 설치합니다.

1. 시작하기
(1) [project CNT] 프로그램을 실행합니다.
(2) 메시지 창이 뜨면 USB 포트에 베이스 장치를 연결하고 [확인] 버튼을 누릅니다.
(3) 만약 [project CNT] 프로그램을 실행하기 전에 이미 베이스 장치를 연결하였다면 장치를 분리하고 프로그램을 종료한 후,
    과정 (1)을 다시 수행하여 주십시오.

2. 경기하기
(1) [센서 ON] 버튼을 눌러 센서입력을 시작합니다. 3분으로 세팅된 타이머가 0초가 되면 자동으로 센서입력이 종료됩니다.
    만약 그 전에 센서입력을 종료하고 싶은 경우, [센서 OFF] 버튼을 눌러 종료할 수 있습니다. 
    만약 3분 후에도 센서입력을 받고 싶은 경우, 다시 [센서 ON] 버튼을 눌러 센서입력을 받을 수 있습니다.
(2) [수정] : 센서입력이 종료된 상태에서, [수정] 버튼을 눌러 <개인 당일 점수> 값을 변경할 수 있습니다.
    [리셋] : 센서입력이 종료된 상태에서, [리셋] 버튼을 눌러 진행중인 경기의 값을 초기화 할 수 있습니다. (단, 이전 경기의 값까지 초기화 되지는 않습니다.)
    [저장] : 센서입력이 종료된 상태에서, [저장] 버튼을 눌러 진행중인 경기의 값을 <ingamedata>에 저장할 수 있습니다. (ingamedata 인게임 데이터는 게임 내부적으로 저장되는 기억공간입니다.
    저장하지 않으면 게임 종료 후 데이터가 기억되지 않으므로 반드시 저장하고 넘어가시기 바랍니다.)

    *주의사항 : 반드시 센서입력이 종료된 상태에서 [수정], [리셋/저장] 버튼을 누르십시오.
(3) [다음] 버튼을 눌러 다음 경기로 넘어갑니다. 이전 경기로 돌아가고 싶을 경우 [이전] 버튼을 눌러 돌아갑니다.
    *주의사항 : [이전] 버튼은 [다음] 버튼에 비하여 내부 로직이 복잡하여 상대적으로 오류가 날 확률이 높습니다. 따라서 가급적 [이전] 버튼 보다는 [리셋] 버튼의 사용을 권합니다.
    [이전] 버튼을 사용하지 않고 과거의 데이터를 수정하고 싶은 경우, <ingamedata> 파일을 통해 값을 직접 수정할 수 있습니다.

    *<< 간단요약  >> : [센서 ON] -> [센서입력 종료됨을 확인] -> [리셋] or [수정] -> [저장] -> [다음]

3. 종료하기
(1) [종료] 버튼을 눌러 게임을 종료합니다. [종료] 버튼을 누르면 <gamedata> 파일에 당일의 경기 성적을 저장 후 종료하므로 
    (X) 버튼이 아닌, [종료] 버튼을 눌러 종료할 수 있도록 합니다.
(2) <data> - <gamedata> 파일을 열어 경기 성적을 확인합니다. 4-2) 를 참고하십시오.

4. 텍스트 파일 설정
1) 플레이어 이름 입력하기
  (1) <data> - <ingamedata> 파일을 열어 플레이어의 이름이 적힌 부분에 실제 이름을 적고 저장합니다.
  (2) 만약, <data> 폴더가 존재하지 않거나 <ingamedata> 파일이 존재하지 않는 경우 한 번 [project CNT] 프로그램을 시작하고 종료하면 자동으로 폴더와 파일이 생성됩니다.
    <ingamedata>는 실제 게임에서 프로그램이 사용할 정보가 저장되는 파일이므로 가급적 이름을 제외한 다른 부분은 수정하지 않도록 합니다.
    만약, <ingamedata> 의 이상 때문에 프로그램이 정상 작동하지 않을 경우 수정한 <ingamedata> 파일을 삭제하고 (1)의 과정을 다시 수행하십시오.

2) <gamedata> 파일 열기
  (1) <data> - <gamedata.csv>를 우클릭 후 [연결 프로그램] - > [메모장] 을 통해 메모장으로 파일을 엽니다. 
  (2) [다른 이름으로 저장] -> [파일 이름 : 당일 날짜.csv (예 : 200101.csv)] -> [파일 형식 : 모든 파일] -> [인코딩 : ANSI] -> [저장]

      *[project CNT] 에서는 UFT-8 유니코드를 통해 글자를 인코딩하는 반면에 엑셀에서는 ANSI를 통해 글자를 인코딩하므로 이에 맞게 인코딩 방식을 변경해야 글자가 깨지지 않습니다.
  (3) [연결 프로그램] -> [엑셀] 을 통해 엑셀로 해당 파일을 엽니다.

3) [project CNT] 에서 한글이 깨질 경우
  (1) <data> - <ingamedata> 열기
      [다른 이름으로 저장] -> [파일 이름 : ingamedata.tsv] -> [파일 형식 : 모든 파일] -> [인코딩 : UFT-8] -> [저장] -> [기존 파일 덮어쓰기]

      *4-2)와 같은 이유로 생기는 문제이므로, 이번에는 ANSI를 UFT-8로 바꾸어줍니다.
